This is the **backend** of a Flood Alert System built using **FastAPI**.  
It receives water level and rainfall data and sends **SMS alerts** to registered phone numbers to warn about potential floods.

---

## Features
- Backend-only system (no frontend)
- Built with **FastAPI** for fast API development
- Sends real-time SMS alerts using an SMS API (like Twilio)
- Can be extended for IoT sensor integration

---

## Technologies Used
- **Python 3**  
- **FastAPI**  
- **Uvicorn** (ASGI server)  
- **Twilio** or other SMS API  
- **python-dotenv** for environment variables
